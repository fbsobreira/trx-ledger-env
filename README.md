# TRX Ledger Dev Environment - TRON.NETWORK

### Installation

```sh
$ git clone https://github.com/fbsobreira/trx-ledger-env.git
$ git submodule init
$ cd trx-ledger-env
$ chmod +x install.sh
$ ./install.sh
```
### Run samples 

```sh
$ cd samples/blue-sample-apps/blue-app-helloworld 
$ make load
```
